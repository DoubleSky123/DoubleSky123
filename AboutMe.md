  ## 🚀 Featured Project

  ### [AI Clinical Assistant](https://github.com
  /DoubleSky123/aiclinical)
  > Ambient clinical documentation powered by
  LangGraph · FAISS · Qwen 2.5 · Whisper

  A full-stack AI system that turns
  physician-patient transcripts (text or audio)
  into structured SOAP notes with ICD-10-CM
  coding, medication validation, and clinical
  risk flagging.

  **Architecture highlights:**
  - 🧠 Multi-agent LangGraph pipeline with
  parallel ICD audit, medication review, and
  risk detection nodes
  - 🔍 MMR retrieval over FAISS vector store
  grounded in clinical guidelines
  - 🎙️  Whisper ASR for real-time audio
  transcription
  - ⚡ Redis caching + Prometheus monitoring
  - 🌐 React chat-style frontend with streaming
  updates
